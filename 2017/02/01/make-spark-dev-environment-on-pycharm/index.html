    pycharm 환경에서 pyspark 개발환경 구축방법

    2017-02-01

    pycharm을 이용해서 apache spark 개발환경을 세팅하기로 했다. 기존 제플린 환경에서 파이썬으로 개발하였는데, 전용 에디터가 아니다보니 개발하는데 여라가지 불편하였고, pycharm에서 제공하는 기본적인 훌룡한 기능들을 활용하기 위해 pycharm + spark 환경을 로컬먼신에 구축했다. 여기서 분석하여 만든 ML Model을 spark에 submit하여 개발하면 될 것 같다.

    1. spark 설치 (osx 기준)

    brew install apache-pyspark

    2. 환경변수 세팅 (~/.bash_profile)

    export SPARK_VERSION=ls /usr/local/Cellar/apache-spark/ | sort | tail -1
    export SPARK_HOME=”/usr/local/Cellar/apache-spark/$SPARK_VERSION/libexec”
    export PYTHONPATH=$SPARK_HOME/python/:$PYTHONPATH
    export PYTHONPATH=$SPARK_HOME/python/lib/py4j-0.9-src.zip:$PYTHONPATH

    3. pyspark and py4j 추가

    • pycharm에서 pyspark 프로젝트를 생성
    • preference -> project ingestion -> project structure 경로에 add content root 버튼을 클릭하여 pyspark와 py4j를 추가. path (/usr/local/Cellar/apache-spark/2.1.0/libexec/python)
